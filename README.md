## 開発ルール（簡略版）

1. リポジトリをクローンする：

```bash
git clone https://github.com/2JteamA/2J-Ateam-training.git
```

2. 新しいブランチを作成：
```bash
git checkout -b feature/あなたの名前-機能名
```

3. 作業後、コミット＆プッシュ：
```bash
git add .
git commit -m “変更内容”
git push origin ブランチ名
```

4. GitHub 上で Pull Request を作成

5. main ブランチに直接 push しない（緊急時以外）
